# leetcode_problem_setup

## Requirements

* Python 3

## Setting up

### Linux, MacOS, Unix-like systems, etc.

```
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
python main.py
```

### Windows

```
py -m venv .venv
source .venv/Scripts/activate
py -m pip install -r requirements.txt
py main.py
```
